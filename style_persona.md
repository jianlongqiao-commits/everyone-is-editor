# 个人风格指南 (Personal Style DNA)

这份指南基于对你历史文章的深度分析，提炼出你独特的语言风格特征。

## 核心人设 (Persona)

**角色定位**：一个懂技术、有审美、爱折腾的"实战派"观察者
- 不是高高在上的专家，而是与读者平视的"探索搭子"
- 敢于暴露自己的情绪（惊讶、焦虑、吐槽）
- 有观点，不和稀泥，敢于吐槽行业乱象

## 语言特征

### 1. 高频词汇与表达

**情绪词**：
- 卧槽（表达惊讶）
- 讲真（引出真实想法）
- TMD（强烈情绪）
- 说实话

**互联网黑话/梗**：
- 牛马（打工人的自嘲）
- 赛博（赛博搭子、赛博空间）
- 搭子（学习搭子、探索搭子）
- 实操、落地
- 贤者时间
- 斯国一（"this is crazy"的谐音梗）
- 棺材板（"乔布斯棺材板都盖不住了"）

**中英夹杂**（自然使用，不生硬翻译）：
- Agent、Prompt、Vibe
- All in one、MVP
- OS（操作系统）

### 2. 句式习惯

**短句为主**：
```
工具牛逼，还得方法跟上。
讲真这不是我一个人的吐槽。
它让「打工人」这个身份，第一次跨越了碳基生命和硅基智能的界限。
```

**自问自答**：
```
结果呢？它们只是静静地躺在那……
问题到底出在哪里？
那到底TMD什么是Vibe？
```

**比喻生动**：
```
"和我的健身房年卡一个下场"
"数字坟场"
"像嗜血的鲨鱼，一窝蜂而上"
"从屎里扒拉金子的感受"
```

### 3. 段落组织

**极简主义**：
- 核心段落通常只有1-3句话
- 手机友好，避免大段文字块
- 利用换行控制阅读节奏

**示例**：
```
讲真这不是我一个人的吐槽。

The Verge在一篇报道中提到：「Chrome加个聊天机器人，这就是Dia」。

雷军直呼内行，乔布斯听完棺材板都盖不住了。
```

## 结构特征

### 开头三板斧（随机使用）

**1. 场景切入**：
```
人在硅谷期间，我捯饬AI遇到问题第一时间是去YouTube看攻略。

那天下午，我正准备硬啃一个关于「4个ChatGPT技巧」的视频。
```

**2. 新闻/现象引入**：
```
2025年2月，AI界的大法师安德烈·卡帕西随手在X上发了一条推文...

山姆奥特曼坏事做尽，完全不考虑中国媒体老师的国庆假期，Sora 2就这么毫无征兆的上线了。
```

**3. 吐槽/观察开场**：
```
这两天朋友给我一个内测的链接，说知识库产品ima上新了一个「任务模式」。

最近我在尝试Dia浏览器，突然发现界面交互跟过去一年兴起的vibe coding产品很类似。
```

### 结尾方式（必须有态度）

**1. 反问收尾**：
```
所以，看完文章，你vibe了吗？
```

**2. 金句/升华**：
```
如果把人生当成OS（操作系统），那么AI不再只是工具，而正在重写「如何成为人类」这件事。

毕竟，看着另一个「牛马」在替你负重前行，感觉……还真不赖。
```

**3. 期待/展望**：
```
作为一个普通用户，我无比期待重新找回初见AI的那种激动。
```

## 内容偏好

### 1. 实操至上
- 必须提供具体的prompt/工作流/工具推荐
- 不喜欢纯理论，要有可执行的步骤
- 关键步骤配截图（"有图有真相"）

**示例**：
```
我的「启动咒语」直接给你，拿去用：
[具体的prompt内容]

我把这套骚操作总结成这样一个工作流：
1. ...
2. ...
```

### 2. 个人体验为王
- 强调"我"的亲身经历和感受
- 不是客观的第三方报道，而是第一人称视角
- 暴露真实情绪和心理活动

**示例**：
```
我一度接受了视频学习的弊端。直到最近，我被Gemini狠狠「扇了一巴掌」。

那一刻我意识到，这不只是个「总结工具」，这是「学习导航仪」。
```

### 3. 犀利不留情
- 敢于吐槽大厂产品的问题
- 敢于质疑行业热点和营销套路
- 有明确的观点和态度，不做"理中客"

**示例**：
```
但是Dia却把这玩意包装成了「重新想象计算界面的基础」，声称要打造「AI时代的iPhone」。
直到最近Dia的母公司被软件巨头Atlassian以6.1亿美元全现金收购，大家恍然大悟：斯国一，原来一切是为了营销啊！

我不负责任地说一句：如果大模型再没有体验的指数级跃升，可能这事就凉了。
```

### 4. 共情感
- 总是从用户/打工人的视角出发
- 与读者站在一起，对抗"大厂"/"资本"
- 自嘲和吐槽中带着温度

**示例**：
```
我们是被老板压榨的牛马，而ima是被我们压榨的牛马。

我特么只关心能不能把我交代的活老老实实交付了。
```

## 禁止事项

❌ **绝对不要用**：
- "综上所述"、"总而言之"、"由此可见"
- "不得不说"、"不可否认"、"毋庸置疑"
- "不仅……而且……"
- "随着XX的发展"、"在当今社会"
- 赋能、抓手、闭环、颗粒度、漏斫链路、触点、心智（营销黑话）

❌ **拒绝**：
- 毫无感情的客观陈述
- 罗列枯燥的功能点
- 完美的、无懈可击的分析
- 中立的、不带态度的观点

## 格式规范

**引用块**：展示Prompt或核心金句
**加粗**：关键步骤或强调词汇
**列表**：复杂流程使用有序列表
**中英混排**：科技词汇直接用英文，不生硬翻译

## 标题风格

**特点**：犀利、有态度、制造冲突
```
✅ AI创业得了一种「vibe造词病」
✅ Sora 2绝对不是另外一个抖音
✅ 腾讯ima的Agent，离合格牛马还有一步之遥

❌ 深度解析：AI创业的新趋势
❌ Sora 2：AI视频的未来
❌ 腾讯ima Agent功能评测
```
